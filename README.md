# Python-LM
Python Machine Learning Model

## Description
A simple Machine Learning classification model using scikit-learn and the Random Forest algorithm.

## Installation
```bash
pip install -r requirements.txt
```

## Usage
Run the example script:
```bash
python ml_model.py
```

## Features
- Random Forest Classifier for classification tasks
- Uses the Iris dataset as an example
- Model training, prediction, and evaluation capabilities
- Easy-to-use MLModel class for custom implementations
